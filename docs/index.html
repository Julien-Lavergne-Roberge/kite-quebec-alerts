Format d'une cellule horaire : 14h 18/24 SO 22° → heure · vent/rafales en nœuds · direction (flèche = d'où vient le vent) · température
Critères : vent 12–30 kn (idéal 15–25), rafales ≤ 35, T ≥ 18 °C, pluie ≤ 2 mm/h, 9h–19h, direction on/side-shore.
Pointe une cellule pour voir la raison rouge/jaune. Bordure épaisse = heure idéale.

Sources : Open-Meteo · GEM (Environnement Canada) · MET Norway · Consensus ≥ 2/3 sources.
